About the site

To be honest, I never liked the blogging features in social media such as Deviantart's journals or tumblr, the customization options are either too limiting, behind a pay wall or non-existing.

And as a fan of everything of the 90s and mid 2000s, I decided to make my website from the ground up (programming included!).

My goal with this project is to have a little space on the web where I can share whatever I want in form of nine bibles of text like people used to do in the 2000s.
And what better way to do that than in a fully customizable, old-school website!

Because I want to be transparent with you, and assure you that the site is secure. I have to mention that this site is stored in a github github repository using github pages service, here's a link to said repository if you want to chech out by yourself.
